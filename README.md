# BukuGajah
Another rip-off app for learning purposes! :sunglasses:

## Installation

```javascript
npm install
```

## Requirements

This project uses global nodemon and webpack. Please install them globally on ur machine :)

## Run Dev Server

```javascript
npm run dev
```

## Run Production Server

```javascript
npm run prod
```

## API Test Suites

```javascript
npm run test-api
```

## React Test Suites

```javascript
npm run test-react
```
